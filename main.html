计算公式

complete值 = 分数 ÷ (物量 × 2400)

ACC奖励值（分段函数）：

0% ≤ ACC < 90%：ACC奖励值 = 0%

90% ≤ ACC < 95%：ACC奖励值 = 1.31% × (ACC - 90)%

95% ≤ ACC < 98%：ACC奖励值 = 6.56% + 1.97% × (ACC - 95)%

98% ≤ ACC < 99%：ACC奖励值 = 12.46% + 2.62% × (ACC - 98)%

99% ≤ ACC < 99.5%：ACC奖励值 = 15.08% + 3.28% × (ACC - 99)%

99.5% ≤ ACC ≤ 100%：ACC奖励值 = 16.72% + 6.56% × (ACC - 99.5)%

Performance Score = complete值 + ACC奖励值 ÷ 100

注意：ACC奖励值是百分比，需要除以100后再与complete值相加。
